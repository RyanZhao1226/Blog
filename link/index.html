Awesome Links and Resources

Hello, dear visitors! Welcome to my personal website. I am truly grateful for your visit and would like to share some of my favorite resources that have helped me grow in the field of computer science. Here is a diverse collection of friendly links and resources that I find useful, educational, and entertaining. I hope they will benefit you as well.

Table of Contents

  1. Online Learning Platforms
  2. Programming Languages
  3. Development Tools
  4. Competitive Programming
  5. Open Source Projects
  6. Blogs and Podcasts
  7. Community Forums

Online Learning Platforms

  1. Coursera - A great platform for online learning, offering a wide range of computer science courses from top universities.
  2. edX - Another fantastic platform providing various CS courses from prestigious institutions.
  3. Codecademy - An interactive and beginner-friendly platform for learning programming languages.

Programming Languages

  1. Python - The official website for Python, a versatile and powerful programming language.
  2. Java - The official website for Java, a widely-used and popular programming language.
  3. JavaScript - Mozilla Developer Network’s documentation for JavaScript, the backbone of web development.

Development Tools

  1. Visual Studio Code - A lightweight and powerful source code editor with built-in support for various programming languages.
  2. GitHub - A platform for version control and collaboration that helps developers work together on projects.
  3. Docker - A containerization platform that simplifies the deployment of applications.

Competitive Programming

  1. LeetCode - A popular platform for practicing coding problems and preparing for technical interviews.
  2. Codeforces - A competitive programming platform featuring contests, problems, and a strong community.
  3. HackerRank - A website for practicing coding challenges and learning about various domains.

Open Source Projects

  1. TensorFlow - An open-source library for machine learning and artificial intelligence, developed by Google.
  2. React - A popular JavaScript library for building user interfaces, developed by Facebook.
  3. Linux - The official website for the Linux operating system, a powerful and open-source OS.

Blogs and Podcasts

  1. Joel on Software - A blog by Joel Spolsky, covering various aspects of software development.
  2. Coding Horror - A blog by Jeff Atwood, discussing software development, programming, and technology.
  3. Software Engineering Daily - A daily podcast on various software engineering topics, featuring interviews with industry experts.

Community Forums

  1. Stack Overflow - A question-and-answer platform where developers can find solutions to their programming problems.
  2. Reddit - The /r/learnprogramming subreddit, a helpful community for learning programming and getting advice.
  3. Dev.to - A platform where developers share articles, tutorials, and ideas on various programming topics.

Online Books and References

  1. Eloquent JavaScript - A comprehensive book by Marijn Haverbeke that teaches JavaScript programming from scratch.
  2. You Don’t Know JS - A book series by Kyle Simpson, diving deep into the core mechanisms of JavaScript.
  3. Real Python - A collection of in-depth tutorials, articles, and resources for learning Python programming.

YouTube Channels

  1. Corey Schafer - A YouTube channel offering tutorials on Python, web development, and more.
  2. Traversy Media - A channel providing web development tutorials, from HTML and CSS to JavaScript frameworks.
  3. Computerphile - A YouTube channel with videos on various computer science topics, featuring explanations from experts.

I hope you find these links and resources useful in your computer science journey. If you have any suggestions or want to share your own favorite resources, please feel free to contact me. Happy coding!
